# AI English Teacher - Legal Pages

Legal and support pages for the AI English Teacher iOS app, hosted via GitHub Pages.

## GitHub Pages Deploy Steps

1. Create a new public GitHub repository (example: `ai-english-teacher-legal`).
2. Upload all files in this folder to that repo root.
3. In GitHub repo: `Settings` → `Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main` and folder: `/ (root)`.
6. Save and wait 1-2 minutes.

## Your URLs will be:

- **Privacy Policy:** `https://<username>.github.io/ai-english-teacher-legal/privacy-policy.html`
- **Terms of Use:** `https://<username>.github.io/ai-english-teacher-legal/terms-of-use.html`
- **Security Policy:** `https://<username>.github.io/ai-english-teacher-legal/security-policy.html`
- **Support:** `https://<username>.github.io/ai-english-teacher-legal/support.html`

## App Store Connect Settings

Use the Privacy Policy URL in:
1. **App Store Connect → App Privacy → Privacy Policy URL**
2. **App Store Connect → App Information → Privacy Policy URL**

Replace the old Google Docs link with:
```
https://<username>.github.io/ai-english-teacher-legal/privacy-policy.html
```
